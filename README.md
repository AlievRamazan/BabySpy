## ğŸ’¡ The Solution

Babyspy is a compact, wearable sensor that pairs with a smartphone application.

### How It Works
1.  **The Sensor:** A small, safe device with no sharp edges attaches to the child's clothing.
2.  **The App:** Parents load a map of their house into the Babyspy smartphone app.
3.  **Safety Zones:** Parents mark "Dangerous Areas" on the map (e.g., open windows, electrical outlets, kitchen stoves).
4.  **Real-Time Alerts:** Using Bluetooth technology, if the sensor detects the child is entering a marked dangerous zone, the parent's phone rings immediately.

> **Smart Integration:** The system is designed to communicate with other smart home sensors (e.g., window sensors) to not only alert parents but potentially trigger safety mechanisms (like automatic window closing).
